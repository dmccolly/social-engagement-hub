üìã Basic Embed Code (Recommended)

Copy and paste this code into your Webflow page or any HTML page:

<!-- Social Engagement Hub Widget -->
<iframe 
    src="https://gleaming-cendol-417bf3.netlify.app/widget/newsfeed?headerColor=%2310b981&headerText=Community%20Feed&maxPosts=10&theme=light"
    width="100%"
    height="800"

    title="Community Feed"
></iframe>

üé® Customization Options

You can customize the widget by modifying the URL parameters:

headerColor - Header background color (hex code, URL encoded)
Example: %2310b981 (green) or %233b82f6 (blue)
headerText - Header text (URL encoded)
Example: Community%20Feed or Latest%20Updates
maxPosts - Number of posts to display
Example: 5, 10, 20
theme - Color theme
Options: light or dark
showAvatars - Show user avatars
Options: true or false
showInteractions - Show like/reply buttons
Options: true or false
showCreateButton - Show "Share an Update" button
Options: true or false
borderRadius - Corner roundness in pixels
Example: 0, 8, 16
